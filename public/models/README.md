# サンプルBlenderモデル

このフォルダには、Blenderで作成した3Dモデル（.glb形式）を配置してください。

## 配置例:
- character.glb - キャラクターモデル
- building.glb - 建築物モデル  
- vehicle.glb - 乗り物モデル
- furniture.glb - 家具モデル

## ファイル命名規則:
- 英数字とハイフン（-）のみ使用
- 拡張子は .glb または .gltf
- 例: my-robot-character.glb

## ファイルサイズの推奨:
- 5MB以下（ウェブ表示の快適性のため）
- 大きなファイルは圧縮またはポリゴン数削減を検討

詳細な手順については、プロジェクトルートの BLENDER_GUIDE.md を参照してください。
